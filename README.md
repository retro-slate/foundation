# RetroSlate Foundation

RetroSlate Foundation is a global initiative focused on improving the health of people, animals, and the environment through data, collaboration, and action.

This repository contains the foundation platform, including:

- Interactive regional map
- Global Health Index (GHI)
- Organization and project profiles
- Public reports
- Events and volunteer opportunities
- Admin-managed data system

## Tech Stack

### Frontend
- Next.js
- TypeScript
- Tailwind CSS

### Backend
- Django
- Django REST Framework
- PostgreSQL
- Supabase

### Infrastructure
- Vercel
- Render
- Supabase
- Resend
- PostHog
